Saving and Sharing Data

Explore efficient methods for saving and sharing your TLC data, including exporting options, recommended file formats, and tips for seamless collaboration.

Step-by-Step Instructions

  1. Access the Quantity App. Currently, only this part of the app allows you to copy and paste model data with predefined settings for easy sharing.

  2. Copy the model data from the "Copy-paste model data" input field and paste it into the same input field in the application on another device. The tool automatically applies the settings upon pasting.

  3. When focusing on the input field, the content is updated dynamically. This ensures you are working with the most recent data.

  4. To collaborate with peers, share the exported file via email or a secure file-sharing platform. Ensure all participants have access to the required tools for viewing and editing the data.

Tips for Effective Data Sharing

Follow these best practices for efficient and reliable collaboration:

  • Always use a consistent file naming convention to avoid confusion.
  • Include a short description or metadata file explaining the content of the exported data.
  • Ensure that all collaborators are familiar with the tool and file formats used for exporting data.
